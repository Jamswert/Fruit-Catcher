# Fruit Catcher

A game where you catch fruit that fall from the sky in a basket and avoid rotten / bomb fruits.

## Overview

Fruit Catcher is an arcade-style game built with Pygame. The objective is simple: catch as many fruits as possible in your basket while avoiding rotten fruits and bombs that fall from the sky.

## Game Objective

The aim of the game is to get as many points as possible by catching good fruits and avoiding dangerous items.

## Requirements

- Python 3.x
- Pygame

## Installation

1. Clone this repository
2. Install the required dependencies:
   ```bash
   pip install pygame
   ```

## Running the Game

Run the game using:
```bash
python main.py
```
